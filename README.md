# Polyantha

An experimental interpreted functional programming language,
that is meant to be simple and readable.
It is mostly an hobby project so don't expect much from it.

## Installation

```bash
git clone https://github.com/Milepowe/Polyantha.git
cd Polyantha
sbt build
```

Then add the resulting binary in your path. Run this to verify that it's correctly installed

```bash
polyantha --version
```

## Usage

Run an example:

```bash
polyantha examples/helloworld.antha
```
